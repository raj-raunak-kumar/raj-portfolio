# **App Name**: Evolving Hero

## Core Features:

- Omnitrix UI: Interactive Omnitrix-themed UI to select different persona modes.
- Dynamic Content Rendering: Render content based on the selected persona mode (e.g., Research, Skills, Timeline, Contact).
- Energy Background Simulation: Simulate an Omnitrix-style energy field as the background, dynamically adapting to the selected mode.

## Style Guidelines:

- Primary color: Ben 10 Green (#39FF14) for UI highlights, accents, and Omnitrix-related elements.
- Background color: Dark Green (#006400), a desaturated, darker shade of the primary color to set the appropriate mood.
- Accent color: Electric Lime (#CCFF00). Used to denote interactive elements such as hover states on buttons.
- Body font: 'Rajdhani', a sans-serif font with a modern, sporty feel. 
- Headline font: 'Orbitron', a geometric sans-serif with a science fiction aesthetic. 
- Use 'lucide-react' icons, styled with the primary and accent colors, to represent different sections and actions.
- Use a full-screen layout with distinct sections for each mode, and a fixed header for mode selection.
- Subtle fade-in animations when transitioning between sections and modes to enhance the user experience.
